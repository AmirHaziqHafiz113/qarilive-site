Flow Steps

Each card shows who is involved + what happens
Step 1 Customer
Customer starts the purchase
Customer is interested and is ready to buy via an Agent/Master Agent referral.
Step 2 Customer Master Agent WhatsApp / QR
Customer opens the Master Agent page
Customer clicks a WhatsApp link to the Master Agent page or scans a QR code that leads to it.
Step 3 Customer Master Agent Xendit Checkout
Customer pays via Master Agent Xendit page
Payment is completed on the Master Agent’s checkout page (this ties the sale to the Master Agent).
Step 4 Agent Customer Screenshot Proof
Agent requests purchase screenshot
Agent asks customer for the payment screenshot to redeem the “2 Years Free Content” benefit.
Step 5 Agent Admin Master Agent
Agent sends proof + Master Agent reference to Admin
Agent forwards the screenshot to Admin and includes the Master Agent reference code so commissions can be calculated correctly.
Step 6 Admin Transaction Check
Admin verifies and confirms purchase
Admin checks the transaction (e.g., Xendit transaction ID/amount) and confirms it is valid before payout.
Step 7 Admin Master Agent Bank Transfer
Admin transfers Master Agent commission
Admin pays the Master Agent commission based on the confirmed transaction and reference.
Step 8 Admin Agent Bank Transfer
Admin transfers Agent commission
Admin pays the Agent commission after the purchase is confirmed.
Minimal checks (recommended)
• Keep a unique Master Agent reference code to avoid payout disputes.
• Admin should verify transaction ID + amount before any transfer.
• Store screenshot + reference code for audit.
